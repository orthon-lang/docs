## Philosophy

Orthon is built around a simple idea:

> Everything is data. Everything else is a transformation of data.

The language minimizes special syntax and favors a small set of orthogonal primitives that can be composed together.

Instead of introducing dedicated language features for every concept, Orthon represents them as data modifiers.
